User guide

Open 3 or more terminals and direct to this folder

To start the server type 'go run server.go'
Then type 'go run client.go' and 2 or more terminals

You can now join the chatroom by entering a name in the client terminal
Once joined is it possible to send messages to other users in the chatroom
If you wish to leave type '/leave', this will inform others in the chatroom that you left, aswell as stop the client program
